# README Generator
This project is a CLI tool for generating complete README files. It is built in node using Inquirer.

## Installation
1. Clone the repo
```
git clone git@github.com:rjwrightme/CLI-README-generator.git
```
2. Navigate to the repo and run
```
npm i
```
